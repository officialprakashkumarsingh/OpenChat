# OpenChat Environment Variables Reference
# Template for setting up environment variables in Vercel deployment

## Required Environment Variables

### Search API Keys
BRAVE_API_KEY=your_brave_search_api_key
TAVILY_API_KEY=your_tavily_search_api_key  
EXA_API_KEY=your_exa_search_api_key

### Google OAuth Authentication
AUTH_GOOGLE_ID=your_google_oauth_client_id
AUTH_GOOGLE_SECRET=your_google_oauth_client_secret

### Convex Backend Configuration
NEXT_PUBLIC_CONVEX_URL=your_convex_deployment_url
CONVEX_DEPLOYMENT=your_convex_deployment_string

## Optional Environment Variables

COMPOSIO_API_KEY=your_composio_api_key

NEXT_PUBLIC_POSTHOG_KEY=your_posthog_project_key

## Deployment Instructions

1. **Go to Vercel Dashboard:**
   - Navigate to your project settings
   - Find "Environment Variables" section

2. **Add Required Variables:**
   - Copy each variable name (left side of =)
   - Add your actual values (not the placeholders shown here)
   - Set environment to "All" (Production, Preview, Development)

3. **Redeploy Application:**
   - Go to Deployments tab
   - Click "Redeploy" on latest deployment

## Security Notes

- Never commit actual API keys to version control
- Use environment variables in deployment platform
- Rotate keys regularly for security
- Use different keys for development/production environments
